### Perintah Termux :
    $ apt update -y && apt upgrade -y
    $ pkg install git
    $ pkg install python3
    $ git clone https://github.com/dafidxcode/BotFB
    $ cd BotFB
    $ pip install -r requirements.txt
    $ python bot_fb.py

### Update Script :
    $ rm -rf $HOME/BotFB
    $ git clone https://github.com/dafidxcode/BotFB
    $ cd BotFB
    $ python bot_fb.py
